1. FLIGHT BOOKING DATA CLEANER
--------------------------------------------------------------------------------

Problem:
Flight information was provided as raw strings in the format:
    "SOURCE-DESTINATION:PRICE"

Some price values were invalid or corrupted.

Approach:
- Split each string into route and price
- Extract source and destination cities
- Convert price into Number format
- Validate invalid price values using isNaN()
- Replace invalid prices with 0
- Filter flights within a required price range
- Sort flights from cheapest to highest price
- Convert final output into JSON format

Concepts Used:
- split()
- Number()
- isNaN()
- map()
- filter()
- sort()
- JSON.stringify()

3. MOVIE STREAM ANALYTICS
--------------------------------------------------------------------------------

Problem:
Movie details were provided as unstructured strings:
    "MOVIE_NAME|GENRE|VIEW_COUNT"

The goal was to identify highly viewed Action and Sci-Fi movies.

Approach:
- Parsed movie strings into structured objects
- Converted view count into Number format
- Validated corrupted view data
- Filtered Action and Sci-Fi movies
- Selected movies with views greater than 5000
- Sorted movies by highest views first
- Returned final data as JSON

Concepts Used:
- String Parsing
- map()
- filter()
- sort()
- JSON.stringify()
